Quick Commands

Terminal Specific Commands

  • cd ..
  • cd documents
  • cd documents/archive
    • Or you can start with "cd" and then drag folder over
  • ls

Git Specific Commands

  • git clone http://myrepo.here
  • git status
    • use this to check the status of repo compared to local copy
  • git add index.html
    • this simply stages your doc saying you're ready to commit now
  • git commit -m "adding index.html"
    • git commit actually commits the document and -m allows you to add a message
  • git push
    • after committing, git push will upload your file to the repo on github
  • git pull
    • to pull down latest changes from git hub
